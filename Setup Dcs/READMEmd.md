# n8n GCP Deployment Documentation

## Overview
This repository contains modular, step-by-step guides for building and maintaining an n8n instance on Google Cloud Platform (GCP).  
Each `.md` file is a self-contained tutorial with dependencies, commands, and version metadata.

This format supports:
- Continuous updates as GCP and n8n change
- Reuse across written docs and YouTube tutorials
- Simple linking between related modules
- Obsidian or MkDocs compatibility
